# DJ Rills Official Website

## Overview
This is the official website for **DJ Rills (Mr. Mambo Mbaaya)**, showcasing his events, mixes, and contact information. The site is designed to be modern, visually appealing, and mobile-friendly, providing fans with an easy way to follow his work and book him for events.

---

## Features
- **Hero Section:** Engaging welcome section with DJ Rills branding and tagline.
- **About Section:** Overview of DJ Rills’ musical style and appearances.
- **Events & Festivals:** Showcases past and upcoming events with Instagram recap links.
- **Mixes Section:** Embeds DJ Rills’ latest mixes via YouTube.
- **Testimonials:** Highlights fan feedback and experiences.
- **Contact Form:** Allows users to book DJ Rills for events or send inquiries.
- **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
- **Smooth Animations:** Fade-up effects for sections when scrolling.

---

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome** for icons
- **Responsive Grid Layouts**
